# Lubaba Binta Abubakar - Personal Portfolio

This is a **personal portfolio website** built with **HTML** and **Tailwind CSS**. It is a single-page, responsive, and modern design to showcase personal skills, projects, and contact information.

---

## 🌟 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices.
- **Hero Section**: Full-screen introduction with name, profession, and call-to-action.
- **About Section**: Short bio and list of skills.
- **Portfolio Section**: Showcase projects with images and links.
- **Contact Section**: Contact form and social media links.
- **Footer**: Minimal footer with copyright information.
- **Built with Tailwind CSS**: Fully customizable utility-first framework.

---

## 🛠️ Technologies Used

- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
- HTML5 & CSS3
